Ejercicio del Día (equipo: Facundo Zampetti - Gonzalo Rossi - Ignacio Griboff)
==============================

Cada equipo debe llegar con el servidor listo y accesible por SSH.
Además, deben poder abrir el servidor desde VS Code y trabajar directo en el VPS.

Checklist técnico

**1- Acceso al servidor**

- Usuario creado con permisos de sudo.
- Acceso por SSH funcionando.

**2- VS Code conectado por SSH**

- Extensión Remote - SSH configurada.
- Conexión exitosa al servidor.

**3- Herramientas base instaladas en el VPS**

- Git
- Docker y Docker Compose

**4- Stack backend listo para iniciar**

- FastAPI creado en contenedor.
- MySQL y phpMyAdmin en Docker.
- Variables de entorno y red Docker básica.

**5- Seguridad inicial**

- JWT para autenticación.
- Hashing de contraseña con bcrypt.


## Entregable (Capturas y explicación)

- Acceso SSH al servidor funcionando.
- VS Code conectado por SSH.
- Captura o evidencia de Docker funcionando.
- Proyecto FastAPI inicial en el servidor.
- Contenedores MySQL y phpMyAdmin levantados.
- Documento del proyecto (borrador) en el repo.
